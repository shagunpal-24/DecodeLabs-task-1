# Responsive Portfolio Website

## 📌 Project Overview

This project is a modern and responsive portfolio website built using HTML, CSS, and JavaScript. The website is designed to provide a clean and professional user experience while showcasing personal information, skills, projects, and contact details.

The project focuses on responsive web design principles, smooth navigation, modern UI components, and mobile-friendly layouts.

---

## 🚀 Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Project/Service Cards
- Contact Form
- Footer Section
- Mobile Responsive Design
- Smooth Scrolling Navigation
- Modern UI Design
- Flexbox & CSS Grid Layouts

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
